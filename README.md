
## Installation

```bash
$ npm install
```

```bash
$ docker-compose up --build -d
$ docker exec -it mysql_db mysqk -u academy -p

docker build -t tu_usuario_docker/nombre_de_tu_imagen .
docker login
docker push tu_usuario_docker/nombre_de_tu_imagen

```
## MYSQL
 SHOW DATABASES;

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
